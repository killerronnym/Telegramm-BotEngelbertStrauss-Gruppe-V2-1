So funktioniert's

Der Filter überwacht alle eingehenden Nachrichten (in allen Gruppen, wo der Bot Admin ist).

  • Findet er ein Wort von der Blacklist in einer Nachricht, wird die Nachricht sofort gelöscht.
  • Er erteilt dem Nutzer automatisch eine rote Karte (Verwarnung über /warnings).
  • Wird die maximale Anzahl an Warnungen erreicht, greifen die Bestrafungen (Mute, Kick, Ban) aus den Einstellungen des ID-Finder Bots.
