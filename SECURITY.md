# Sicherheitsrichtlinie (Security Policy)

## Unterstützte Versionen

Nur die neueste Version wird mit Sicherheitsupdates unterstützt. Bitte stellen Sie sicher, dass Sie immer die aktuellste Version der Bot Engine verwenden.

| Version | Unterstützt         |
| ------- | ------------------- |
| v1.9.50 | :white_check_mark:  |
| < 1.9.50| :x:                 |

## Melden einer Sicherheitslücke

Haben Sie eine Sicherheitslücke gefunden? Wir schätzen Ihre Hilfe, um dieses Projekt sicher zu halten.

1. **GitHub Issues**: Sie können Sicherheitslücken direkt auf GitHub unter "Issues" melden.
2. **Prüfung**: Wir werden Ihre Meldung umgehend prüfen und feststellen, ob sie in der nächsten Version behoben werden kann.
3. **Behebung**: Je nach Schweregrad der Sicherheitslücke kann die Veröffentlichung einer Korrektur einige Tage oder Wochen dauern.

Vielen Dank für Ihren Beitrag zur Sicherheit dieses Projekts!
